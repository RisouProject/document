将来欲しいプラグインの書き方

前提#

  • VSCode (要領がわかるなら他のエディタでも良い)
  • GitHub Desktop (要領がわかるなら他のGitクライアントでも良い)

書き方#

まずはドキュメントをダウンロードしてくる#

  1. GitHub Desktopを開き、Clone a repository from the Internet...を選択する
  2. 検索バーにRisouProjectと入力し、RisouProject/documentを選択する
  3. その下にあるバーで、どこにダウンロードするかを選択・確認する

ドキュメントを作成する#

  1. VSCodeを開く
  2. 左上のファイル(F)から、フォルダーを開くを選択する
  3. さきほどクローンしたリポジトリのフォルダを選択する
  4. 左のエクスプローラーにあるdocsを押して展開する
  5. plugin-featuresを右クリックし、新しいファイルを選択する
  6. 追加されたファイルに適当なファイル名をつけて、Enterを押す

必要な情報を書き加える#

  1. 先程作ったファイルをクリックして、中身を開く
  2. 以下の必要な情報を書く
---
id: そのドキュメントの英語名
title: そのドキュメントの日本語名　と　そのプラグインの欲しい度と優先度を合わせた評価値
---

例

---
id: tax
title: 税金 ★10
---

あとは、他のドキュメントと見比べて、真似して書く！
