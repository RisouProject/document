## コマンド

| コマンド  | 説明       |
| -------- | --------- |
| `/risou bank`  | 1のメニューを開く |


## 前提
- RisouTax
- RisouMoney (というかVault)


## システム
流石に経済系はだるいかな

RisouMENUでRisouBANKをクリックするとチャット欄にフォーマットを出す。

## フォーマット

### 1 - メニュー
```
ーーーRisou Bankーーー
お取引を選択してください。
[お預入れ]
[お引出し]
[お振込み]
[残高照会]
[国庫残高]
ーーーーーーーー
```

[]内をクリックするとフォーマットが表示。


### 2 - 預け入れ
```
ーーーRisou Bankーーー
預入れ手数料は無料です。
お預入れ金額をチャット欄にお書き下さい。
ーーーーーーーーー
```

チャット欄に入力した額をBANKに預ける。


### 3 - 引き出し
```
ーーーRisou Bankーーー
引出し手数料は２００Rです。
お引出し金額をチャット欄にお書き下さい。
ーーーーーーーーー
```


### 4 - 振り込み
```
ーーーRisou Bankーーー
振込手数料は無料です。
最初に取引相手のIDをチャットにお書きください。
ーーーーーーーーー

ーーーRisou Bankーーー
次に振込額をチャット欄に入力してください。
ーーーーーーーーー

ーーーRisou Bankーーー
振込が完了しました。
ーーーーーーーーー
```


### 5 - 残高照会
```
ーーーRisou Bankーーー
残高：○○R
このまま取引を続けますか？
[預入れ] [引出し] [振込]
ーーーーーーーーー
```


### 6 - 国庫
```
ーーーRisou Bankーーー
皆様の税金によりこの世界は成り立っています。
国庫残高：○○R
ーーーーーーーーー
```


## 問題点
銀行に対応している税金システムがない。
- 税金は他鯖をみればわかる通り絶対必要なもの。だが銀行に預けることで脱税が容易にできてしまう。
- よって所持金額＋銀行残高金額－税金にしないといけない。
