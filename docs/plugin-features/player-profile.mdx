## コマンド

| コマンド  | 説明       |
| -------- | --------- |
| `/good [プレイヤー名]`     | プレイヤーに高評価    |
| `/bad [プレイヤー名]`      | プレイヤーに低評価    |
| `/profile [プレイヤー名]`  | プロフィールを表示    |
| `/profile edit`      | プロフィールをカスタム |


## 利点
こういうのが好きな人は一定層いる。開きやすくすれば称号の為にRisou Advancementが賑わう。他の鯖にないシステムで差をつけられる。モチベ維持。


## システム
デフォルトは1.00 Like1回で＋0.75 Bad1回で−0.50
評価値が0.70下回ったら一時BANでDiscordの解除申請に書き込んで貰って、スタッフがBadが不適切って思ったらPardon


## 問題点
荒らしが適当にマイナス押して回る可能性

- クールダウンは12時間。1人1回までで、3人まで評価できる。
- 評価値が2.0以上じゃないと評価を押せなくする。 難しそう。
