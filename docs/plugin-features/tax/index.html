徴税 ★10

欲しい度

★★★★★

優先度

★★★★★

コマンド#

特になし

システム#

RisouBANKと提携して欲しすぎて禿げそう。

下記類似プラグインはログイン時間で税金ではなく、現実時間で税金なので鯖にいなくてもどんどん金が吸われてしまう。

しかも国庫がないので、国庫アカウントを用意しないといけない。→国庫アカウントを用意しなくても国庫を作りたい。

ログインしたときに、税金何円取られました、みたいなのを表示したい。

所持金額＋銀行残高金額から税金⇒ログイン時間関係なく１週間に一回

１時間に稼いだ額から〇%税金を一時間ごとに取る→これはログイン中のみ ※稼がなければ税金は０円！

集めた税金はRisou BANKの国庫に収納できれば完璧... 所持金額の合計ごとにグループ分けは必須

  • 破産者：合計額が0R以下⇒毎時間の税率０％・合計額からとる税率0%
  • 貧困層：合計額が0~50000⇒毎時間の税率３．０％・合計額からとる税率0.5%
  • 一般層：合計額が50001~300000⇒毎時間の税率４．０％・合計額からとる税率1.5%
  • 一般上位層：合計額が300001~500000⇒毎時間の税率５．０％・合計額からとる税率2.0%
  • 準裕福層：合計額が500001~1000000⇒毎時間の税率６．０％・合計額からとる税率3.0%
  • 裕福層：合計額が1000001~1500000⇒毎時間の税率７．０％・合計額からとる税率3.5%
  • 超裕福層：合計額が1500001以上⇒毎時間の税率８．０％・合計額からとる税率5.0%

※取った税金で宝くじしたり、何かテーマを決めて建築してお金配布するのもよさそう。
